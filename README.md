# nuxt-md-cards

Proof of Concept of nuxt using:
* pug templating
* vuetify
* dropdown target route
* pouch database
* Sync Pouch to remote couchdb
* progressive web app

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
